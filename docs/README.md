# Documentação — CONSIGTEC

Pasta para documentação do projeto: roadmap, especificações, decisões de
arquitetura e materiais de referência.

## Sugestão de organização

- `roadmap.md` — roadmap / fases / backlog do produto
- `arquitetura.md` — visão técnica (stack, integrações, fluxo git ↔ Base44)
- `especificacoes/` — specs por módulo/área
- `decisoes/` — registro de decisões (ADRs)

> Suba aqui os arquivos do roadmap. Assim que estiverem no `main`, o time
> (e o assistente) consegue acompanhar o progresso contra o roadmap oficial.
